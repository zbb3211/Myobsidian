- 通过cnt来记录数组个数
- 数组独特记录数据方式
- 以及用数字来分辨两个不同组别的数据
```c++
int cnt = 0;
scanf("%d", &n);
while (n -- )
{
	int x, y;
	scanf("%d%d", &x, &y);
	int d = sqr(x) + sqr(y);
	if (d <= distance) w[cnt ++ ] = {x, y, d, 0};
}

scanf("%d", &m);
while (m -- )
{
	int x, y;
	scanf("%d%d", &x, &y);
	int d = sqr(x) + sqr(y);
	if (d <= distance) w[cnt ++ ] = {x, y, d, 1};
}

int res[2] = {0};
for (int i = 0; i < cnt; i ++ )
{
	if (i && w[i].team != w[i - 1].team) break;
	res[w[i].team] ++ ;
}
```
用`cnt ++` ,运行完后便可知道`w[N]`中究竟存了多少数 ；
此外，还用`0, 1`来记录了不同类别；
`i && w[i].team != w[i - 1].team`，判断不是首位与两种不同类别的交接处;
用`res[2] = {0}`，来记录两组数，`w[i].team = 0\1`;
